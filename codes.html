3D Processing:
  1. PCL: A standalone, large scale, open project for 2D/3D image and point cloud processing.
  2. Meshlab: An open source, portable, and extensible system for the processing and editing of unstructured 3D triangular meshes.
  3. VCG library: The Visualization and Computer Graphics Library (VCG for short) is a open source portable C++ templated library for manipulation, processing and displaying with OpenGL of triangle and tetrahedral meshes.
  4. Scanalyze: A system for aligning and merging range data
  5. ScanView: A system for remote visualization of scanned 3D models
  6. Volfill: A hole filler based on volumetric diffusion
  7. VripPack: Volumetric Range Image Processing Package
  8. ZipPack: Polygon Mesh Zippering Package
  9. Mesh Toolbox @ Carnegie Mellon University.
  10. OpenMesh @ Prof. Leif Kobbelt
  11. NormFet Software @ Tamal Krishna Dey, The Ohio State University
  12. ply2vri etc. @ University of Washington
  13. Point Cloud Library (PCL)
  14. ISO2MESH: A 3D surface and volumetric mesh generator for MATLAB/Octave
  15. TetView: A Tetrahedral Mesh and Piecewise Linear Complex Viewer.
  16. FastRBF Interpolation Toolboxes: A Fundamental Advance in Scattered Data Modeling
  17. 3D Object Converter: A powerful 3d polygon model translation and interactive viewing shareware tool that allows arbitrary 3D model data to be imported from external sources, exported to various industry standard 3D file formats with great accuracy and quality. The program supports 615 file formats
  18. Atangeo Balancer SDK: A binary library that allows you to easily integrate Balancer simplification technology into your own tools or plugins for other software.
  19. MEPP - 3D MEsh Processing Platform: basic mesh processing, simplification,curvature analysis, boolean operations,segmentation,progressive compression/watermarking,visual quality metrics,remeshing, mathematical morphology, mesh repairing...
  20. Geomview: An interactive 3D viewing program for Unix.
  21. Tools for Mesh Processing @ University of Washington
  22. Manifold Mesh Processing: 1) Simple smoothing/noise reduction, 2) Bare-bones point cloud to mesh and robust normals for point clouds, 3) Mesh simplification, 4) Curvature calculation on meshes, 5) Local shape descriptors on meshes.
  23. Plyview: a simple program that's used to view Cyberware PLY (.ply) file format polygonal meshes.
  24. Blender: open source 3D graphics and animation software
  25. Princeton Vision & Robotics Toolkit (PVRT): an open-source software library including a diverse set of functions that are useful and non-trivial to implement for fast-prototyping in vision and robotics research.
3D Data Producer:
  1. OpenMVG: "open Multiple View Geometry" is a library for computer-vision scientists and especially targeted to the Multiple View Geometry community.
  2. PMVS: Patch-based Multi-view Stereo Software @ University of Washington
  3. Bundler: Structure from Motion (SfM) for Unordered Image Collections @ University of Washington
  4. MVE: Multi-View Environment @ TU Darmstadt. It supports the development of algorithms based on multiple views, such as Multi-View Stereo or Photometric Stereo
  5. Computer Graphics Research Software: A good collection.
